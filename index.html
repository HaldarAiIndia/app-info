// Features
Everything in one place

Stop digging through Android settings. App Info surfaces every detail about every app — instantly.

📦
Package Browser
Browse all installed apps with package names, version codes, and install dates in a clean searchable list.
📊
Usage Statistics
See exactly how long and how often each app is used. Spot your most active apps at a glance.
💾
Storage Breakdown
Know the exact size of every app — APK, data, and cache — so you can reclaim space quickly.
🔐
Permissions Viewer
See every permission each app holds. Know which apps access your camera, contacts, or location.
🏅
Certificates & Signatures
View signing certificate details and verify app authenticity before trusting any APK.
⚡
Quick Settings Jump
Jump straight to any app's Android system settings page with a single tap from within App Info.
// How it works
Three taps to full visibility

No learning curve. App Info is designed to surface information the moment you need it.

1
Install & open App Info
Download the app and grant the usage access permission when prompted. This lets App Info read app statistics on your device.
2
Browse your installed apps
All apps appear instantly in an alphabetical list with their package IDs. Use the search bar to find any app in milliseconds.
3
Tap any app for full details
Open Basic Information, Usage Statistics, Permissions, or Certificates & Signature — everything about that app, organized and readable.
// Transparency
What we ask and why

App Info only requests permissions it genuinely needs. Here's exactly what each one does.

QUERY_ALL_PACKAGES
Lists all installed apps so they appear in the browser. Without this, the app list would be empty.
GET_PACKAGE_SIZE
Reads APK, data, and cache sizes for each app so you can see storage breakdowns.
PACKAGE_USAGE_STATS
Reads how often and how long apps are used. Requires your explicit approval in Android Settings.
INTERNET
Used for app updates and optional analytics. Your app list is never uploaded to any server.
POST_NOTIFICATIONS
Lets App Info send you update alerts. You can disable notifications at any time in Android Settings.
